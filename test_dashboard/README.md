# Test Automation Dashboard

## Overview
A Flask-based dashboard to visualize test execution results, coverage metrics, and UI failure screenshots.

## Features
- **Summary Stats**: Total tests, Passed, Failed, and Coverage %.
- **Run History**: Line chart showing pass/fail trends over time.
- **Failure Analysis**: List of failed tests with error messages.
- **Screenshots**: Gallery of screenshots captured during failed Selenium UI tests.
- **Run Trigger**: Execute tests directly from the dashboard.

## Setup
1. **Install Dependencies**:
   ```bash
   pip install flask pytest-json-report
   ```

2. **Run Dashboard**:
   ```bash
   python test_dashboard/app.py
   ```

3. **Access**:
   Open [http://localhost:5001](http://localhost:5001) in your browser.

## Data Sources
- `report.json`: Generated by `pytest --json-report`.
- `coverage.json`: Generated by `coverage json`.
- `history.json`: Maintained by the dashboard app (appended on each run).
- `tests/ui/*.png`: Screenshots generated by `tests/ui/conftest.py` on failure.
